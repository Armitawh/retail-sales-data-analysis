# Sales Analysis and Visualization

## About
This project analyzes lemonade sales data using Python and visualizes trends and relationships. It uses **pandas** for data manipulation and **matplotlib / seaborn** for plotting.

## Features
- Summary statistics: mean, median, standard deviation
- Histograms of sales distribution
- Scatterplots of sales vs. temperature
- Barplots comparing average sales by weather condition
- Correlation analysis between temperature and sales

## Files
- `sales_analysis.py` - Main Python script with EDA and visualization
- `README.md` - Project description
